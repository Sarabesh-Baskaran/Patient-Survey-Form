# 🏥 Patient Survey Form

A simple and interactive patient survey form built using HTML, CSS, and JavaScript, integrated with Firebase Firestore to store patient data in real-time.

---

## 🚀 Features

* 📋 User-friendly patient data collection form
* ✅ Form validation for required fields
* 🔥 Real-time data storage using Firebase Firestore
* 🎯 Clean and responsive UI
* ⚡ Instant success feedback after submission

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend (BaaS):** Firebase Firestore
* **Tools:** Firebase Console, Git, GitHub

## ⚙️ Setup & Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/patient-survey-form.git
   ```

2. Navigate to the project folder:

   ```bash
   cd patient-survey-form
   ```

3. Open `index.html` in your browser
   (or use Live Server in VS Code)

---
